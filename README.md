## Projects

**WiSense — WiFi-Based RF Object Detection** &nbsp; `Eclipse Hackathon 6.0`

Camera-free presence detection through WiFi Channel State Information (CSI) distortion. Built in 24 hours. Pipeline: CSI signal → 2D heatmap → Canny edge detection → SVM classifier (29 features). Validated against the Widar3.0 dataset.

`Python` `NumPy` `SciPy` `OpenCV` `scikit-learn` `Streamlit` &nbsp;·&nbsp; [Live demo](https://wisense-b4dfjicsgagn3v5coabpaq.streamlit.app/) &nbsp;·&nbsp; [Repository](https://github.com/code-rh11Reyhan/WiSense)
